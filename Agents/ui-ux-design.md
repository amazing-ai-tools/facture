# UI/UX Design - facture

Use this skill for product screens, visual design, and interaction work.

## Current UI
- One responsive landing/status screen
- Hero section with production URL and BugZero dashboard links
- Three provisioned capability cards
- Plain CSS, no design system yet

## Principles
- Replace placeholder copy with actual product value as soon as the domain is known.
- Keep layouts responsive from 320px upward.
- Use accessible buttons, links, focus states, and semantic structure.
- Use `lucide-react` icons when icons help scanning.
- Avoid decorative complexity until the product workflow is clear.
