## Routing
- Product ambiguity: PM and CEO
- User interface: Frontend and UI/UX Design
- Build or deploy failure: Deploying and Infra
- Browser behavior: Frontend Testing and QA
- Secrets, widget, dependency, or CI risk: Security Audit
- API/server need: Architecture Design and Backend
