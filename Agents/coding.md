# Coding - facture

Use this skill for implementation work in the `facture` repository.

## Stack
- React 18
- TypeScript with strict compiler settings
- Vite 5
- Plain CSS in `src/styles.css`
- `lucide-react` icons

## Rules
- Read the relevant file before editing it.
- Keep changes scoped to the requested behavior.
- Preserve `ensureBugZeroWidget()` unless the task explicitly changes bug reporting.
- Use typed React code; avoid `any`.
- Run `npm run build` after meaningful code changes.
- Do not add a backend, router, state library, or component framework without a product reason.
